# Vacation Home Reservation System (Internet Applications Programming Project)

## Description

This project is a web-based **Vacation Home Reservation System**, developed as part of the **Internet Applications Programming course** at the School of Electrical Engineering, University of Belgrade.

The system allows users to browse, search, and reserve vacation homes through a frontend application connected to a backend API and a database.

---

## Features

- Search and view available vacation homes
- User registration and authentication
- Make and manage reservations
- Frontend + backend communication
- Persistent storage using a database
- Add and delete vacation homes
- etc.

---

## Technologies

- Frontend: Angular / HTML / CSS 
- Backend: Node.js / Express
- Database: MongoDB
